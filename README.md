# **Network Administration Internship – Week 02**

## **Overview**

This repository contains my **Week 02 internship tasks** focused on IP subnetting and Virtual LAN (VLAN) implementation. The work was completed using both **Cisco Packet Tracer** and **GNS3** to strengthen practical networking skills through hands-on configuration and testing.

---

## **Objectives**

* Practice **Fixed Length Subnet Mask (FLSM)** subnetting.
* Solve **Variable Length Subnet Mask (VLSM)** subnetting problems.
* Design and analyze different subnetting scenarios.
* Configure **Virtual LANs (VLANs)** in both GNS3 and Cisco Packet Tracer.
* Verify network connectivity through appropriate testing.

---

## **Tasks Completed**

### **1. FLSM & VLSM Exercises**

* Solved multiple FLSM subnetting problems.
* Solved VLSM subnetting exercises with different network requirements.
* Designed various subnetting scenarios and assigned IP addresses according to host requirements.
* Calculated network addresses, broadcast addresses, subnet masks, and usable host ranges.

### **2. VLAN Configuration**

* Implemented VLANs in both **GNS3** and **Cisco Packet Tracer**.
* Created a network consisting of:

  * **2 Switches**
  * **8 PCs**
  * **2 VLANs**
* Configured:

  * VLAN creation
  * Access ports
  * Trunk link between switches
  * End-device IP addressing
* Each VLAN contains **4 PCs**.
* Verified communication between devices within the same VLAN.
